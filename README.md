## Ouvrir le projet localement

Double-cliquez sur `index.html`. Les styles, les images, le JavaScript et les
versions française, anglaise, arabe et japonaise fonctionnent sans serveur.

Pour utiliser la version Flask dynamique, lancez depuis la racine du projet :

```powershell
python -m pip install -r requirements.txt
python app.py
```

Ouvrez ensuite `http://127.0.0.1:5000`. Copiez `.env.example` vers `.env`
pour définir une clé de session propre à votre environnement.
